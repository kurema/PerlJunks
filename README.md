# PerlJunks
Perlのちょっとしたスクリプト集です。  
2009年頃に作ったものです。

## htmlEx
コメント内に配置した命令で比較的簡単に動的ページを作るCGIツールです。  
目指した所はよりシンプルなphpでした。

当時は頑張って作りましたが相当ひどいので実用には適しません。  
今ならXMLの適当な名前空間を命令と解釈する形にすると思います。

index.cgiは一部正常に動作しません。

## nico2srt
ニコニコ動画の投稿者コメントをsrtファイルに変換するスクリプトです。

## fs
DVDの容量いっぱいにファイルを組み合わせるだけのスクリプトです。  
短いですし、なんてことはないですが結構使いました。  
全探索なのでファイル数が多いと重いですが、案外普通に使えます。

## mml2number
ニコニコ大百科のピコカキコで使われているMMLを数字の羅列に変換するスクリプトです。
